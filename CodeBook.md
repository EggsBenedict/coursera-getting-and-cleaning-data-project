# Code Book

This code book summarizes the resulting data fields in `tidydata.txt`.

## Identifiers

* `subject` - Subject ID
* `activityName` - Type of activity being performed when the corresponding measurements were taken. See below for list of activities.

## Activity Name

* `LAYING`
* `SITTING`
* `STANDING`
* `WALKING_DOWNSTAIRS`
* `WALKING_UPSTAIRS`
* `WALKING`
